按应用定制策略

Xcode 强制英文、微信强制中文、终端保持原状 — 每个 App 独立配置，一次设定永久生效。

⚡

乐观缓存切换

TIS 切换成功后立即更新本地缓存，消除异步读取竞态，确保每次切换 App 输入法都正确响应。

🛡️

隐私优先

无需辅助功能权限，基于 NSWorkspace 通知感知焦点切换，不读取任何键盘输入内容。

🔍

应用选择器

从全部已安装应用中搜索并选择，支持按名称和 Bundle ID 过滤，不局限于当前前台应用。

🔔

HUD 实时提示

切换时屏幕右上角短暂显示当前输入法（简体中文 / English），操作反馈清晰直观。

⌨️

CapsLock 兼容

通过模拟系统级 CapsLock 事件完成切换，保留 macOS 原生 CapsLock 切换中英文的能力。

技术实现

TIS 乐观缓存

切换成功后直接写入目标状态，避免 TIS 异步生效期间的缓存错误，彻底修复切换失效问题。

事件合并去重

NSWorkspace 激活通知经 8ms DispatchWorkItem 合并，过滤同 App 重复事件，快速 Cmd+Tab 更低延迟。

懒构建菜单

NSMenuDelegate 实现菜单内容仅在点击时渲染，消除每次 App 切换时的 UI 主线程压力。

快速安装

1

下载 Tail-Input-1.3.1.dmg，打开后将 App 拖入 Applications

2

首次运行，点击菜单栏 ⌨️ 图标确认正常

3

为常用 App 设置策略，开启 开机自启动，搞定
